# Financial Data Analysis: Mean-Variance-Standard Deviation Calculator

### Project Overview
This project is a high-performance statistical calculator built using **Python** and the **NumPy** library. It was developed as part of the **freeCodeCamp Data Analysis with Python Certification**. 

The tool accepts a list of nine digits, transforms them into a 3x3 matrix, and calculates the mean, variance, standard deviation, max, min, and sum along both axes and for the flattened matrix.

### Professional Context
As an **Accounting & Finance professional**, I developed this project to demonstrate the application of computational tools to traditional financial reporting tasks. By leveraging NumPy’s multi-dimensional arrays, this script provides the foundation for automating:
* **Variance Reconciliation:** Rapidly calculating deviations across different reporting periods.
* **Data Integrity:** Implementing custom error handling (`ValueError`) to ensure data sets meet specific reporting requirements.
* **Financial Modeling:** Moving beyond Excel to more scalable Python-based analytical frameworks.

### Technical Features
* **Matrix Manipulation:** Efficiently reshapes data using NumPy arrays.
* **Statistical Analysis:** Multi-axis calculations (Rows, Columns, and Flattened).
* **Robust Error Handling:** Validates input length to ensure data consistency.

### How to Run
1. Ensure you have Python 3 and NumPy installed.
2. Import the `calculate` function from `mean_var_std.py`.
3. Pass a list of 9 integers to the function to receive a structured dictionary of results.
